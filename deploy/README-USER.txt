Here are the instructions for installing the app from the “.tar folder”:

The app is packaged with a version of the R software in a zipped folder. This will only work on a Windows computer. To run the app you will need to:

1. Download the “caribou_demography_app.tar” from [here](https://can01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fgithub.com%2FLandSciTech%2FCaribouDemographyBasicApp%2Freleases%2Ftag%2Fv0.0.0.9002&data=05%7C02%7CDanielle.Cantin%40ec.gc.ca%7C63a4bf827df946f1d2dd08dda2ab7886%7C740c5fd36e8b41769cc9454dbe4e62c4%7C0%7C0%7C638845579557725475%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C0%7C%7C%7C&sdata=ongVSKpl0k8NuLXvcbFMhxsE5NsKoZdUjbIXptlLzvw%3D&reserved=0) to your computer.

2. Unzip the file by right clicking on “caribou_demography_app.tar” and selecting "Extract all".

3. Click "Browse" in the dialog box that opens to choose a location to store the app files, then click "Extract".

4. Go to the location where you extracted the files and double click on the file "run.bat".

This will open a Windows command prompt window. Just leave that window open and wait, the app will open in your default internet browser within a few minutes.

The Windows command prompt window will close when you close the browser window.
