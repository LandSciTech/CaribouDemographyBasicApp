# Run the Caribou Demography App

To run the app:

  1. Download caribou_demography_app.tar to your computer.
  2. Unzip the file by right clicking on caribou_demography_app.tar and 
     selecting "Extract all".
  3. Click "Browse" in the dialog box that opens to choose a location to store
     the app files, then click "Extract".
  4. Go to the location where you extracted the files and double click on run.bat
  
The app should open in your default internet browser.
